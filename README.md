# nvim
## 使い方
### プラグイン

| プラグイン | 説明                                                       |
|:----------:|:-----------------------------------------------------------|
| NERDTree   | F2: NERDTreeを開く <br> F3: NERDTreeを閉じる               |
| commentary | gcc: 行をコメント <br> gc + motion: motionをコメントアウト |

### Git

| コマンド | 説明                      |
|:----------:|:------------------------|
| ,ga  | git add .                     |
| ,gc  | git commit .                  |
| ,gsh | git push                      |
| ,gll | git pull                      |
| ,gs  | git status                    |
| ,gb  | git blame                     |
| ,gd  | git vdiff                     |
| ,gr  | git remove                    |
